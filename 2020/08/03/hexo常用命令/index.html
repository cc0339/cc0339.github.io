hexo常用命令

发表于 2020-08-03 更新于 2020-08-06

总结了一些hexo的平时要用到的命令

部署到服务器

hexo clean
hexo generate
hexo deploy

本地调试

hexo generate #生成
hexo server #启动本地服务，进行文章预览调试

常用

hexo new "postName"           #新建文章
hexo new page "pageName"      #新建页面
hexo generate                 #生成静态页面至public目录
hexo server                   #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
hexo deploy                   #将.deploy目录部署到GitHub
hexo help                     # 查看帮助
hexo version                  #查看Hexo的版本

复合命令

hexo deploy -g  #生成加部署
hexo server -g  #生成加预览

命令简写

hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
